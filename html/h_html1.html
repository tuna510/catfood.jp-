■ナビバー リンク横並びにする？検討
初心者の為のCSS:コピペでOK！横並びナビメニュー10選
コピペでできる！cssとhtmlのみで作るメニュー
●html

<nav>
    <ul>
        <!-- 現在位置に当たる部分のliにクラスつけてcssで色つけとくと分かりやすいよ -->
        <li><a href="任意のurl">こんな</a></li>
        <li><a href="任意のurl">感じで</a></li>
        <li><a href="任意のurl">欲しい数だけ</a></li>
        <li><a href="任意のurl">並べる</a></li>
    </ul>
</nav>

●CSS

nav ul {
    /* 画面端からの幅 */
    margin: 0 50px;
    /* 縦 */
    line-height: 〇〇px;
}

nav li {
    /* 要素を縦から横に */
    display: table-cell;
    /* 幅 */
    padding: 0 10px;
    min-width: 50px;
    /* 要素を中央に */
    text-align: center;
}

nav a {
    /* 下線消す */
    text-decoration: none;
    /* 文字の色 */
    color: #555;
}

nav li.current {
    /* 選択中のやつに色つけたりできる */
}

nav li:hover {
    /* マウス乗せたときの変化 好きなようにすればいいと思う borderとかbackground-colorとか */
}





■囲み枠
おしゃれなボックスデザイン（囲み枠）のサンプル30
●html

<div class="box">

    <a href="飛ばしたいリンク">
    <div class="box-yeah">
        <p>名前</p>
    </div>
    </a>

    <!-- 好きなだけ並べる -->

</div>

<!-- 次の段 -->
<div class="main-box">
    <!-- また並べる -->
</div>

●CSS

.box {
    /* 余白 */
    margin: 高さ auto(幅);
    /* 中央寄せ */
    text-align: center;
}

.main-box-all {
    /* 横並びにする */
    display: inline-block;

    height: 幅;
    width: 高さ;
    余白とか
    padding: 任意;
    margin: 任意;
    text-align: center;
    background: 文字色;
    
    影
    filter: drop-shadow(0 3px 3px rgba(176, 186, 187, 0.747));
}

/* あとはhoverとか装飾とかすきなように */

■マウスを乗せたら画像をちょっと拡大
●css
                
/* HTMLには画像とか設定しておく */
.クラス名 {
    width: 幅;
    height: 高さ;
    /* 大きくしたの戻す(割合、速さ) */
    transform: scale(1.0);
    transition-duration: 0.5s;
}

/* ほわってさせる */
.クラス名:hover {
    /* 大きくする(割合、速さ) */
    transform: scale(1.2);
    transition-duration: 0.3s;
}



■stepsでアニメーション いろんなアニメーションがあるのであとでぼちぼちまとめていきます
●html

<!-- 画像はcssで指定するのでここには貼らなくて良い -->
<div class="クラス名"></div>

●css

/* 今回は幅1120px,7枚の画像を右→左に配置してる */
.クラス名 {
position: fixed;
width: 幅;
height: 高さ;
leftとかtopとか: 固定したい位置;
padding: 余白;
text-align: center;
background: url(画像のパス) no-repeat;
animation-play-state: paused;
}

.クラス名:hover {
/* 6回切り替えてアニメーション */
animation: 適当な名前 .4s steps(6) forwards;
}

@keyframes 適当な名前(animationでつけたやつ) {
100% {
/* 1120/7→一枚幅160pxを6回動かす→-960px */
background-position: -960px 0;
}
}
