環境構築-VSCode



本体
　Visual Studio Code ：まずはこれを入れよう！これがないと拡張機能はインストできないぞ

つよつよ拡張機能
　open in browser：HTMLファイルをブラウザで開ける
　Beautify ：F1押してbeautifyfileを選択すると見やすいコードにしてくれる
　HTMLHint ：HTMLの記述ミスを波線で教えてくれる
　ESLint ：JSの記述ミスを波線で教えてくれる
　EvilInspector ：全角スペースを教えてくれる。半角スペーズと全角スペース見分けつかないので便利
　vscode-icons ：何の言語のファイルか分かりやすくアイコンで表示してくれる
　Bracket Pair Colorizer ：どこからどこまでがかっこで囲まれてるか教えてくれる
　Japanese Language Pack for Visual Studio Code ：VSCodeを日本語にする 下記でも説明
　python ：どうせpythonもいじるでしょ入れとけ

　拡張機能は好みで突っ込んでいこう



拡張機能のつっこみ方




毎回整え直さなくても自動でフォーマットするための設定




フォルダの持ってき方

ワークスペース内でもいじれる。
flaskのファイルはデスクトップとか分かりやすいところに置いといたほうが好き



色も自由に変えられるし透明度もガーッていじれる
