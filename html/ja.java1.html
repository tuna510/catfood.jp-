■文字列の出力
System.out.println() は文字列を出力せよ、という命令
文字列はダブルクォーテーションで囲み、文末にはセミコロン(;)をつける
System.out.println("Hello muscle!");  結果：Hello muscle!

// 数字を出力するときはクォーテーション要らないしそのまま式も書ける
System.out.println(1 + 5);  結果：6

// 連結
System.out.println("Hello!" + "muscle!");  結果：Hello!muscle!



■データ型
"Hello muscle!" …String型（文字列）　※Sは大文字
1,10,100 …int型（整数）　※iは小文字
1.5、2.4…double型（小数）
他にもいっぱいあるよ！

■変数の定義（2ブロックめの同時に代入するとこよく使いそう）
①データ型の指定
②変数の名前を決める
// ① ②;
int number;
number = 3;

String name;
name = "Goto";

// 値を取り出す場合
int number;
number = 3;
System.out.println(number);  // 結果：3

String name;
name = "Goto";
System.out.println(name);  // 結果：Goto

// 変数の定義と同時に代入（変数は初期化される）
String text = "Hello muscle!";

// 変数を使って連結もできる
String name = "Goto";
System.out.println(name + "さんこんにちは");  // 結果：Gotoさんこんにちは

int number1 = 100;
int number2 = 50;
System.out.println(number1 + number2);  // 結果：150

// 変数の更新
  // まず定義
int number = 3;
  // そして更新
number = 5; // String型も同様
  // int number = 3;
  // int number = 5;
  // にすると、「その変数もうあるやんけ」ってことで定義できなくなっちゃう。

// 自己代入
int x = 3;
x = x + 5 ;
  // intは3+5で8になる　よくあるあれ
  // このあたりは省略形（+=とか*=とか）使っていこう

■型の変換
System.out.println("ちは" + 23 + "歳です");
  // →23が"23"、つまりString型に自動で変換される（自動変換）

int型　System.out.println(5 / 2);  // 結果：2（小数点以下は出力されない）
double型　System.out.println(5.0 / 2.0);  // 結果：2.5
System.out.println(5.0 / 2);  // 結果：2.5（2が2.0に自動変換される）

// 変数使ってるときに別の型に変換したい！
int number1 = 10;
int number2 = 8;
System.out.println((double)number1 / number2);  // 結果：1.25（doubleつけないと1になっちゃう）
