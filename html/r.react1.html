■React
Reactはサイトの見た目の部分を作ることができる。
Facebook、Airbnb、Dropboxみたいな有名どころもReactを使用している。すごいね。
見た目をつくるコードはほとんどHTMLと一緒。ただし、名前はHTMLではなく「JSX」という。
<h1>や<h2>,<div>などが使えるけどいくつか違いがあって、
return内に複数の要素があるとエラーになる。URLつけたらいいかも
また、JSX部分はreturn()の中のみで、全体はjavascript(JS∋JSXなイメージ)
JSXに記述した部分はブラウザに表示される。


■基本
ファイル名App.jsの中に書いていくよ
returnの中がJSX 次からはrenderから書きます
ファイル名App.jsの中に書いていくよ
renderの中がメイン 次からはrenderから書きます
import React from 'react';　// Reactをimport
class App extends React.Component {　// React.Componebtを継承するクラス(今回はApp)の定義 
  render() { {/* JSXを戻り値とするrenderメソッドを定義 */}
    return (
      <h1>Hello World！</h1>
    );
  }
}

export default App;　// 結果：Hello World！ 　// クラスをexport



■エラーが出ないように注意

{/* エラーの出ないコード */}
render() {
    return (
        <div>
            <h1>じゃあどうすればいいかというと</h1>
            <h2>こんな感じでdivとかで囲って</h2>
            <p>一つの要素の中にたくさんつっこむならエラーは出ないよ</p>
        </div>
    );
}

■javascriptの記述
renderメソッドのreturnの外にはjavascriptを書くことができるし、
returnの中(JSX)にもJSを埋め込むことができる。

render() {
    // JSのコメントはJSのときと同じで//をつければOK
    const name="Maria";
    const ingUrl ="画像url";
    return (
        <div>
        {/* 埋め込むときは{}で囲むよ！タグの属性(画像URLとか)も{}を使えばJSを記述できる */}
            <h1>{name}</h1>
            <img src={imgUrl}/>
        </div>
    );
  }
{/* 結果：Maria 指定した画像が表示される */}
