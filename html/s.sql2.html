■データベース(以下db、ダメージボーナスではない)って何？
名前やメールアドレス、テキスト、画像などあらゆるものを保存しておけるもの。
現在のwebサービスにはほぼ100％dbが使用されている。

■なんでいっぱい表作るの？
各テーブルに関係性を持たせて柔軟にデータを作るため。
これを関係性データベース(RDB)と呼ぶ。Excelをつよつよにしたもの。

■dbって何ができるの？
・データを表で管理できる
・検索ができる、条件をつけたりリンクさせたりも
・表を作ったり表にデータ追加、更新、削除ができる
・多くの人が同時に使用できる！すごい！

■dbを使用する流れ
dbの作成
テーブルの作成
データの追加、検索、編集、削除(CRUD)

■SQLって何？
Stuctured Query Languageの略
create　database…dbの作成
create table…テーブルの作成
insert…データの追加
update…データの更新
select…データの選択・検索
delete…データの削除

↑みたいなことができる
人間ではなくpythonがSQLにリクエストを送ってるよ


■PRIMARY KEY(主キー)制約
他と絶対にかぶらない番号。主キーが分かればレコードを絶対に特定できる
読み方はおもき～でもしゅき♡でも好きな方で良い

■主キーの役割
・テーブル検索のスピードを格段に早くする
・テーブルデータの整合性を保証する
主キーを設定した列は同じデータは登録できないためエラーとなり、NULL値も登録できない
